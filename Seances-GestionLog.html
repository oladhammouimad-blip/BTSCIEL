Étape 1 : Remplacer if/else par switch/case

Problème identifié : L'alternative if...else devient très lourde pour traiter l'ensemble des choix de menu.

❌ Avant (if/else)

if (choix == 1) {
    cout << "Option 1";
} else if (choix == 2) {
    cout << "Option 2";
} else if (choix == 3) {
    cout << "Option 3";
} else if (choix == 0) {
    cout << "Quitter";
} else {
    cout << "Erreur";
}

✅ Après (switch/case)

switch (choix) {
    case 1:
        cout << "Option 1";
        break;
    case 2:
        cout << "Option 2";
        break;
    default:
        cout << "Erreur";
}

📖 Concept : Structure switch/case

switch (variable) {
    case valeur1:
        // Instructions pour valeur1
        break;
    case valeur2:
        // Instructions pour valeur2
        break;
    default:
        // Si aucun case ne correspond
}

⚠️ Points critiques

  • break; : Obligatoire après chaque case (sinon exécution continue)
  • default: : Équivalent du else final
  • Types supportés : int, char, enum (pas de string)

🎯 Mission

Transformez votre structure if/else de l'itération 1 en utilisant switch/case. Créez un case pour chaque option de menu (1, 2, 3, 0) et un default pour les erreurs.

Étape 3 : Standards de formatage du code

📝 Règles de formatage C++

1. Indentation

✅ Correct (4 espaces)
if (condition) {
    cout << "Message";
    if (autre) {
        cout << "Autre";
    }
}
❌ Incorrect
if (condition) {
cout << "Message";
if (autre) {
cout << "Autre";
}
}

2. Nommage des variables

✅ Correct (camelCase)
int choixUtilisateur;
string nomFichier;
bool estValide;
❌ Incorrect
int c;
string n;
bool x;

3. Espacement

✅ Correct
if (choix == 1) {
    cout << "Message" << endl;
}

switch (choix) {
    case 1:
        // Instructions
        break;
}
❌ Incorrect
if(choix==1){
cout<<"Message"<<endl;
}
switch(choix){
case 1:
//Instructions
break;
}

🛡️ Include Guards - Protection contre l'inclusion multiple

Les directives #ifndef, #define, #endif empêchent l'inclusion multiple du même fichier .h, ce qui provoquerait des erreurs de compilation.
