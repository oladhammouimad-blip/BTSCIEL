Contexte et Objectifs

Projet professionnel motivant avec progression technique maîtrisée

Contexte Professionnel

L'entreprise Sécuritech Nord, exposée au risque cyber, souhaite développer une application de gestion centralisée de logs pour sensibiliser les collaborateurs aux contraintes de sécurité et centraliser la surveillance pour le RSSI.

Objectif Principal

Découvrir les concepts fondamentaux de la programmation C++ à travers un projet concret et motivant lié à la cybersécurité. Développement itératif avec complexité croissante.

Approche Pédagogique

Maximum de travaux pratiques, minimum de théorie. Apprentissage par la pratique avec validation continue et accompagnement individualisé selon le niveau des étudiants.

Progression Pédagogique Détaillée

4 itérations avec complexité croissante - De la syntaxe de base aux fichiers

🚀

ITÉRATION 1 - BASES

Premier menu fonctionnel

4h
  • Configuration environnement VS Code + MinGW
  • Syntaxe C++ : variables int/string, cout, cin
  • TP : Premier menu avec if/else
  • Test, debug et validation du programme
Production attendue : Programme avec menu basique, gestion des choix utilisateur et validation des erreurs de saisie
🔄

ITÉRATION 2 - BOUCLES

Menu répétitif + switch/case

4h
  • Rappel itération 1 + corrections individuelles
  • Concept boucle do-while vs while
  • Structure switch/case vs if/else
  • TP : Menu permanent avec switch et gestion pause
Production attendue : Application avec boucle principale, structure switch complète et formatage de code professionnel
📂

ITÉRATION 3 - FONCTIONS

Découpage modulaire

4h
  • Pourquoi les fonctions ? Concept de modularité
  • Prototypes vs définitions, paramètres
  • TP : Création fonctions afficherMenu() et choisirLog()
  • Organisation en modules .h/.cpp + tests
Production attendue : Code restructuré avec fonctions organisées, module Menu créé (.h/.cpp) et include guards maîtrisés
📄

ITÉRATION 4 - FICHIERS

Lecture de logs réels

4h
  • Contextualisation métier : importance des logs
  • Manipulation fichiers : ifstream/ofstream, getline()
  • TP : Lecture fichiers de logs + gestion d'erreurs
  • Génération rapports automatisés + statistiques
Production attendue : Application complète de gestion de logs avec lecture/écriture fichiers, rapports de synthèse et analyse de données
